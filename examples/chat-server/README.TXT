Simple MSRP chat server
-----------------------

This is example code for use of the MSRP RA.

Requirements:
------------
1. JBOSS_HOME pointing to Mobicents JAIN SLEE 2.x 
2. Mobicents JAIN SLEE 2.x with SIP11 RA and MSRP RA deployed

Run:
---
1. Install SIP RA and MSRP RA
2. install this example
3. Start Mobicents
4. connect with MSRP clients and start chatting.
